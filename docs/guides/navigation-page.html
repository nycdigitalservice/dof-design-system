Building a Navigation page

A navigation page provides pathways for the user to find the content they’re looking for. Below is guidance for required page markup.

Reset

IMPORTANT: To ensure styles behave as intended, remember to wrap your page elements in the .u-reset class. This can be applied to a main tag.

Header

The .page-header should contain an h1 tag for the page title, and a Grid containing Card elements that link to high-trafficked actions.

Related links

The related links section is another .grid containing .button anchor tags. On some pages these are broken into sections with heading tags.
