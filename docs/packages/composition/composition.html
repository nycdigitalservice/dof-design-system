Container

Inspect code

  margin-inline: auto;
  max-inline-size: calc(var(--nyc-sys-dimension-page-max-width) + var(--nyc-sys-dimension-gutter) * 2);
  padding-inline: var(--nyc-sys-dimension-gutter);
  container: container / inline-size;
}
